# StockKids

어린이를 위한 주식 체험 금융 교육 앱

📅 개발 기간: 2026-01-17 ~ 🚧 개발 중...

## 기획 의도

kids-stock-app은 어린이를 위한 주식 체험 금융 교육 앱입니다.  
실제 돈을 사용하지 않고, 아이가 아는 회사와 캐릭터를 통해  
회사와 경제 개념을 자연스럽게 경험하도록 설계되었습니다.

모바일 퍼스트 UI, 단순한 차트, 캐릭터 보상 구조를 통해  
과몰입을 줄이고 ‘이해 중심의 경험’을 목표로 합니다.

## 주요 기능

- 주식 카드 리스트
- 모바일 퍼스트 UI
- 테마 토글 (키즈 / 부모 모드)

## 기술 스택

- React + TypeScript
- Vite
- styled-components
- React Router

## UI / UX 설계

- 모바일 하단 네비게이션
- 캐릭터 기반 주식 카드
