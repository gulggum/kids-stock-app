# StockKids

어린이를 위한 주식 체험 금융 교육 앱

## 기획 의도

- 초등학생도 금융 개념을 자연스럽게 익히도록 설계
- 실제 돈 없이 사이버 머니로 투자 경험 제공

## 주요 기능

- 주식 카드 리스트
- 모바일 퍼스트 UI
- 테마 토글 (키즈 / 부모 모드)

## 기술 스택

- React + TypeScript
- Vite
- styled-components
- React Router

## UI / UX 설계

- 모바일 하단 네비게이션
- 캐릭터 기반 주식 카드
