# StockKids

어린이를 위한 주식 체험 금융 교육 앱

📅 개발 기간: 2026-01-17 ~ 🚧 개발 중...

## 기획 의도

아이들이 주식을 ‘돈벌이’가 아닌  
‘경험’과 ‘이해’의 관점에서 접할 수 있도록 만든  
어린이용 주식 체험 앱입니다.

실제 돈이 아닌 가상 데이터와 코인을 사용해  
투자 흐름, 가격 변동, 기다림과 선택의 개념을  
게임처럼 자연스럽게 경험할 수 있도록 설계했습니다.

## 주요 기능

- React Router를 이용한 페이지 구성
  - 홈 / 마켓 / 자산 / 커뮤니티 / 캐릭터 / 상점 페이지

- 주식 마켓
  - 모크 데이터 기반 주식 리스트 카드
  - 주식 상세 페이지 (차트 + 아이 눈높이 설명)
  - 하루 1회 구매 제한
  - 구매 전 확인 팝업 제공

- 캐릭터 시스템
  - 캐릭터 아이템 착용/해제 (Context 전역 관리)
  - 아이템 세트 효과 제공
  - 캐릭터 경험치 및 레벨 시스템

- 배지 시스템
  - 행동 기반 배지 획득 (예: 오늘의 한 번, 첫 투자)
  - 배지 획득 시 팝업 알림
  - 캐릭터 페이지에서 획득 배지 누적 표시

- 상점
  - 모크 상품 리스트
  - 구매 전 확인 모달
  - 코인 차감 처리

- 상태 관리
  - Coin / Item / Character / Badge Context 분리 관리
  - localStorage를 통한 상태 유지

## 기술 스택

- React + TypeScript
- Vite
- styled-components
- React Router
- Context API

## UI / UX 설계

- 모바일 하단 네비게이션
- 캐릭터 기반 주식 카드
